# 🌷 Sobre o projeto

Este portfólio nasceu para representar quem eu sou como desenvolvedora.

Mais do que uma coleção de projetos, ele reúne minha trajetória entre **Design**, **UX/UI** e **Desenvolvimento Front-end**, mostrando como essas áreas se complementam na construção de interfaces modernas, acessíveis e agradáveis de utilizar.

Cada detalhe foi pensado para transmitir a mesma sensação que procuro criar nos meus projetos: clareza, leveza e organização.

# 🎨 Design System

Toda a identidade visual foi inspirada na cartela **Verão Claro (Light Summer)**.

A paleta utiliza tons suaves e frios para transmitir leveza sem abrir mão do contraste necessário para uma boa leitura.

O sistema utiliza variáveis CSS e `color-mix()` para manter consistência entre temas, componentes e estados de interação.
