Sobre Mim

Desenvolvimento e design: transformando estética em código! 🌸

Olá! Sou uma futura desenvolvedora de software, atualmente cursando Análise e Desenvolvimento de Sistemas (ADS). Minha jornada com a programação é recente, mas a paixão por transformar ideias em código é intensa e crescente.

Esta nova área é enriquecida por uma base de seis anos de experiência profissional em Design (Raining Design) e uma formação em Marketing Digital. Isso me permite criar soluções que não apenas funcionam, mas que também oferecem uma experiência de usuário excepcional e um apelo visual diferenciado. ☕✨

Tecnologias que uso:

HTML CSS Tailwind Javascript PHP MySQL C Github Figma Photoshop Illustrator
